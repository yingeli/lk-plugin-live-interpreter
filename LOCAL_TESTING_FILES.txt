快速开始
========

方法 1: 使用 Makefile (推荐)
----------------------------
make setup          # 设置环境
vim .env            # 配置凭证
make test-quick     # 验证安装
make run-simple     # 运行示例
make help           # 查看所有命令

方法 2: 使用脚本
----------------
./setup_local_dev.sh              # 设置环境
vim .env                          # 配置凭证
source venv/bin/activate          # 激活虚拟环境
python examples/test_local.py    # 验证安装
python examples/simple_interpreter.py  # 运行示例
